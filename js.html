JavaScript
JavaScript to skryptowy język programowania używany do tworzenia interaktywnych stron internetowych. Wykonywany po stronie klienta w przeglądarce, umożliwia dynamiczną manipulację treścią i reakcję na interakcje użytkownika. W połączeniu z HTML i CSS tworzy kompleksowe aplikacje webowe. Jego wszechstronność obejmuje obsługę zdarzeń, manipulację elementami DOM, komunikację z serwerem za pomocą AJAX oraz programowanie asynchroniczne. JavaScript jest powszechnie stosowany również poza przeglądarkami, dzięki środowiskom takim jak Node.js. Jego rosnąca popularność sprawia, że jest kluczowym narzędziem w dziedzinie front-endu i back-endu webowego.
