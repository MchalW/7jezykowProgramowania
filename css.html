CSS
CSS, czyli Cascading Style Sheets, to język używany w web designie do stylizacji HTML. Definiuje wygląd i układ elementów na stronie, kontrolując kolor, czcionki, odstępy i inne właściwości wizualne. CSS pozwala projektantom jednocześnie utrzymywać spójny wygląd na wielu stronach. Stylizacja może być osadzona bezpośrednio w dokumencie HTML lub przechowywana w osobnym pliku. Dzięki zastosowaniu reguł i selektorów, umożliwia skomplikowane układy i responsywność, dostosowując wygląd strony do różnych urządzeń. To kluczowy składnik trójwarstwowej struktury webowej, obok HTML i JavaScript.
Pełna nazwa Cascading Style Sheets
Data powstania 1996
Użycie Wygląd stron internetowych
Środowiska programowania Każdy edytor tekstu + Przeglądarka (dodawany do pliku HTML)
